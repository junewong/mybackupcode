== 目标:
为了能在我的手持设备上方便地通过英语词根和前缀等学习英语.


== 数据内容:
来自于新东方的词典,分为按前缀和词根等方式来学习英语单词,还带有单词的解释以及单词是如何拆解成为这种词根的;
如果可以,最好还能从其他的来源获取到每个单词的发音以及例句.


== 应用的功能点:
可以分为有几个页面,包括:
# 选择词典,包括是按照前缀来还是词根来的.
# 一个列表页面,显示每个前缀或者词根的列表;
# 点击上面的列表可以进入一个新的列表页面,该页面包括如对应的所有单词的内容;
# 上述列表页面具有筛选功能,可以筛选单词和中文含义以及如何拆解字根的解释.
# 同时还有其他的模式功能,比如随机功能,能随机显示一个单词,先隐藏其他信息,用户点击才能显示完整的解释,这种是用来做测试的.这个功能可以放在首页吧.
