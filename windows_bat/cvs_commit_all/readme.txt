============================================================
[说明]
此脚本用于一键自动提交一个目录下所有的文件到cvs中。
@version 0.1.1
@author junewong


============================================================
[使用方法]
1.编辑当前目录中的Log.txt文件，填写好里面bug、author、log三项内容，用于提交完之后自动导入此内容作为cvs的日志；
2.拖动要提交的文件夹到cvs_commit_all.bat文件上，脚本会自动将里面的所有文件夹添加到cvs中，并将二进制和文本文件分开添加，最后再一起全部提交，提交之后的日志从Log.txt文件中导入。
